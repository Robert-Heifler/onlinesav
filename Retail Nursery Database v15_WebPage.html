How It Solves These Problems

Reorder Report: The Reorder report calculates QtyShort as ReorderPoint minus QuantityOnHand and can filter to only items under reorder, producing an actionable purchase list by supplier and category.
Delivery Report: The Delivery report pulls each SalesOrderId with RequestedDate and full delivery contact and address fields, filtered by requested date range and delivery state, so deliveries can be planned and confirmed from one list.
SalesPaySts Report: The SalesPaySts report groups orders by PaymentStatus and sums line totals per order within a date range, making it easy to see where money is stuck and what was paid, pending, or overdue.
