Quick Start Guide

Get up and running with Access Wizard in just a few minutes.

1

Download the Files

Download both AccessWizard.accdb and Engine.accde files (Free Version or Pro Version).

Go to Download Page
2

Place Both Files in Same Folder

Create a folder on your computer and move both files into it. They must be in the same directory to work properly.

Tip: Create a folder like "C:\AccessWizard\" to keep everything organized.

3

Watch the Tutorial Video

Before starting, watch our short instructional video to see how everything works.

Tutorial video will be embedded here

(YouTube/Vimeo embed)

4

Open AccessWizard.accdb & Start Creating

Open the AccessWizard.accdb file in Microsoft Access and go to the "AI Data Table Generator" form. Follow the on-screen instructions to create your database.

Example prompt:

"I need a database to run my nursery business."

System Requirements

Microsoft Access

2016, 2019, 2021, or 365

Windows 10 or Windows 11
